# BudgetIQ 💰

BudgetIQ is a personal expense tracking web application that helps users manage their daily expenses and analyze spending habits.

## Features
- Add and delete expenses
- Category-wise expense tracking
- Monthly and yearly reports
- Graphical visualization using Chart.js
- User login system

## Technologies Used
- Python (Flask)
- SQLite
- HTML
- CSS
- JavaScript
- Chart.js

## Installation

Clone the repository:

git clone https://github.com/BudgetIQ/BudgetIQ.git

Navigate to project folder:

cd BudgetIQ

Install dependencies:

pip install -r requirements.txt

Run the application:

python app.py

Open in browser:

http://localhost:5000
